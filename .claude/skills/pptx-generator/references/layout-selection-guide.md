# レイアウト選択ガイド

2ステップのフローチャートで最適なレイアウトを選択する。

## Step 1: スライドの役割を判断

```
スライドの役割は?
├── プレゼンの1枚目 → title
├── 章の切り替わり → section
├── 行動喚起(最後のスライド) → cta
└── コンテンツスライド → Step 2 へ
```

## Step 2: コンテンツの性質を判断

```
コンテンツの性質は?
├── 数値が主役(2-3個の指標) → metrics
├── 数値比較・推移(4個以上のデータ) → chart
├── 他者の発言・証言 → quote
├── Q&A形式の情報 → faq
├── 複数項目 x 複数軸の比較 → comparison_table
├── 画像＋説明テキスト → image_with_text
├── 2つの対比・比較 → two_column
├── 3つの並列項目 → three_column
├── 4つの並列項目 → four_column
├── 順序のあるステップ → numbered_list
├── 全体構成の俯瞰 → toc
└── 上記以外 → bullet_points
```

## 判断の優先順位

1. 構造的なスライド(title, section, cta)は最優先で判定
2. データの性質(数値、比較、Q&A)で絞り込む
3. 列数(2/3/4列)で分岐
4. 迷ったら bullet_points をデフォルトとする

## 具体例

| 元資料のコンテンツ | 選択レイアウト | 理由 |
|------------------|-------------|------|
| KPI: 売上120%、利益率15% | metrics | 2-3個の数値指標 |
| 2020-2025年の売上データ | chart | 時系列の数値推移 |
| A社 vs B社の機能比較 | comparison_table | 複数項目x複数軸 |
| 導入の3ステップ | numbered_list | 順序のあるステップ |
| メリット・デメリット | two_column | 2つの対比 |
| 3つのプラン(Basic/Pro/Enterprise) | three_column | 3つの並列 |
| お客様の声 | quote | 他者の発言 |
