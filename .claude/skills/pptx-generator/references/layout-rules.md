# レイアウトルール - 15パターン定義

## 共通ルール

- `---` でスライド区切り
- `<!-- layout: xxx -->` でレイアウト指定
- `# 見出し` = section(h1は自動的にsectionレイアウト)
- `## 見出し` = スライドタイトル
- `### 見出し` = キーメッセージ
- `#### 見出し` = カラムヘッダー(two/three/four_column用)
- レイアウト指定なしのデフォルトは `bullet_points`

---

## 1. title - 表紙

- 背景: 紺色(primary)
- テキスト: 白(text.light)
- `##` = タイトル(44pt)
- `###` = サブタイトル(16pt)

```markdown
<!-- layout: title -->
## プレゼンタイトル
### サブタイトル
```

## 2. section - セクション区切り

- 背景: 薄グレー(background.secondary)
- `#` = セクション名(36pt)
- h1記法で自動的にsectionレイアウトになる

```markdown
# セクション名
```

## 3. toc - 目次

- 番号付き丸アイコン + テキスト
- `##` = タイトル
- `###` = キーメッセージ
- `- 項目` = 目次項目

```markdown
<!-- layout: toc -->
## 目次
### 本日お話しする内容
- 項目1
- 項目2
- 項目3
```

## 4. bullet_points - 箇条書き(デフォルト)

- デフォルトレイアウト
- `##` = タイトル
- `###` = キーメッセージ
- `- 項目` = 箇条書き

```markdown
## スライド名
### キーメッセージ
- ポイント1
- ポイント2
- ポイント3
```

## 5. numbered_list - 手順・ステップ

- 番号バッジ付き
- `1. **タイトル**` + 改行 + 説明文

```markdown
<!-- layout: numbered_list -->
## スライド名
### キーメッセージ
1. **項目1**
   説明文
2. **項目2**
   説明文
```

## 6. two_column - 2列対比

- `####` でカラム区切り
- 左右均等配置

```markdown
<!-- layout: two_column -->
## スライド名
### キーメッセージ
#### 左見出し
左の内容
#### 右見出し
右の内容
```

## 7. three_column - 3列並列

- `####` でカラム区切り

```markdown
<!-- layout: three_column -->
## スライド名
### キーメッセージ
#### 見出し1
内容1
#### 見出し2
内容2
#### 見出し3
内容3
```

## 8. four_column - 4列並列

- `####` でカラム区切り

```markdown
<!-- layout: four_column -->
## スライド名
### キーメッセージ
#### 見出し1
内容1
#### 見出し2
内容2
#### 見出し3
内容3
#### 見出し4
内容4
```

## 9. metrics - 数値・KPI

- 数値を大きく表示
- `- **数値** ラベル` の形式

```markdown
<!-- layout: metrics -->
## スライド名
### 数値で見る成果
- **120%** 売上成長率
- **4.8** 顧客満足度
- **500** 新規顧客数
```

## 10. quote - 引用

- `>` 記法で引用
- `> -- 発言者` で著者

```markdown
<!-- layout: quote -->
## スライド名
> 引用文をここに記載します
> -- 発言者名
```

## 11. faq - Q&A

- `**Q: 質問**` + `A: 回答` のペア

```markdown
<!-- layout: faq -->
## よくある質問
**Q: 質問1**
A: 回答1
**Q: 質問2**
A: 回答2
```

## 12. comparison_table - 比較表

- マークダウンテーブル記法

```markdown
<!-- layout: comparison_table -->
## スライド名
### キーメッセージ
| 項目 | A | B |
|------|---|---|
| 項目1 | 値 | 値 |
| 項目2 | 値 | 値 |
```

## 13. image_with_text - 画像＋テキスト

- `![alt](パス)` で画像指定
- その下に説明文

```markdown
<!-- layout: image_with_text -->
## スライド名
### キーメッセージ
![alt](画像パスまたはURL)
説明文
```

## 14. chart - グラフ

- マークダウンテーブルでデータ指定
- グラフ種別は自動判定またはコメントで指定

```markdown
<!-- layout: chart -->
## 売上推移
### 過去5年で3倍に成長
| 年度 | 売上 |
|------|------|
| 2021 | 100 |
| **2025** | 300 |
```

### グラフ種別の自動判定ルール

- 値が `%` 表記 or 合計が約100 → pie(100%積み上げ棒で実装)
- ラベルが時系列(年/月/Q1等) → bar(縦棒)
- `<!-- chart_type: line -->` → line(折れ線、複数系列対応)
- それ以外 → horizontal_bar(横棒)
- ラベルを `**太字**` にすると強調色で表示

### 複数系列の折れ線グラフ

```markdown
<!-- layout: chart -->
<!-- chart_type: line -->
## タイトル
### キーメッセージ
| 年度 | 売上 | 利益 |
|------|------|------|
| 2021 | 100 | 10 |
| 2025 | 300 | 70 |
```

## 15. cta - 行動喚起

- 背景: 薄グレー(background.secondary)
- `##` = メッセージ
- `###` = 補足説明
- `[ボタンテキスト]` = ボタン

```markdown
<!-- layout: cta -->
## 行動を促すメッセージ
### 補足説明
[ボタンテキスト]
```
