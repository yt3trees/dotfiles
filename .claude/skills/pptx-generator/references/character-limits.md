# 文字数制限ガイド

## 文字幅計算ルール

- 全角文字 = 2
- 半角文字 = 1
- 超過時は末尾を「...」で切り詰め

```python
def char_width(text: str) -> int:
    width = 0
    for ch in text:
        if unicodedata.east_asian_width(ch) in ('F', 'W'):
            width += 2
        else:
            width += 1
    return width
```

## キーメッセージのルール

- 体言止め禁止(必ず述語で終える)
- 20-40文字が目安
- 具体的な数値・固有名詞を含める

## 超過時の対応

1. テキストが制限を超過している場合は、まず要約を試みる
2. 要約でも収まらない場合はスライドを分割する
3. 分割不可能な場合のみ末尾を「...」で切り詰める
